# 🖨️ 列印設定指南

本文件說明如何設定瀏覽器以獲得最佳列印效果。

## 問題說明

列印時可能會遇到以下問題：
- ❌ 出現頁首頁尾（網址、日期、頁碼）
- ❌ 出現空白頁面
- ❌ 內容被裁切或縮放

## 解決方案

### Chrome / Edge 設定

#### 1. 開啟列印對話框
點擊「列印訂單」按鈕，或使用快捷鍵：
- **macOS**: `Cmd + P`
- **Windows**: `Ctrl + P`

#### 2. 設定列印選項

在列印預覽視窗中，按照以下步驟設定：

```
┌─────────────────────────────────────┐
│  更多設定 ▼                         │
├─────────────────────────────────────┤
│  ☑ 背景圖形                         │
│  ☐ 頁首和頁尾      ← 取消勾選！    │
│  邊界: 無          ← 選擇「無」     │
│  縮放: 100%        ← 保持 100%      │
└─────────────────────────────────────┘
```

**重要設定：**
1. ✅ **背景圖形**：勾選（確保 QR Code 和圖片正常顯示）
2. ❌ **頁首和頁尾**：取消勾選（移除網址和日期）
3. ⚪ **邊界**：選擇「無」（移除頁面邊距）
4. ⚪ **縮放**：保持 100%（避免內容變形）

#### 3. 儲存設定

- 設定完成後，Chrome 會記住這些設定
- 下次列印時會自動套用相同設定
- 只需第一次設定一次即可

### Firefox 設定

#### 1. 開啟列印對話框
- **macOS**: `Cmd + P`
- **Windows**: `Ctrl + P`

#### 2. 設定選項

```
┌─────────────────────────────────────┐
│  更多設定                           │
├─────────────────────────────────────┤
│  列印背景: 是                       │
│  頁首和頁尾: 無    ← 選擇「無」     │
│  邊界: 無          ← 選擇「無」     │
└─────────────────────────────────────┘
```

### Safari 設定

#### 1. 開啟列印對話框
`Cmd + P`

#### 2. 展開選項
點擊「顯示詳細資訊」

#### 3. 設定
- 邊界：最小
- 背景：勾選「列印背景」
- 頁首頁尾：無

## 快速檢查清單

列印前請確認：

- [ ] 紙張大小已設定為模板尺寸（預設 100mm × 50mm）
- [ ] 頁首頁尾已關閉
- [ ] 邊界設為「無」
- [ ] 背景圖形已啟用
- [ ] 縮放比例為 100%
- [ ] 預覽中只看到訂單卡片，沒有其他內容

## 進階技巧

### 1. 建立自訂列印設定檔

Chrome 允許你為不同的印表機儲存不同的設定：

1. 選擇印表機
2. 設定好所有選項
3. 點擊「列印」
4. Chrome 會記住這個組合

### 2. 使用鍵盤快捷鍵

加快列印流程：
1. `Cmd/Ctrl + P` - 開啟列印
2. 確認設定
3. `Enter` - 開始列印

總共只需 2 秒！

### 3. 批次列印

如果需要列印多張：
1. 在列印對話框中設定「份數」
2. 或在訂單列表頁面選擇多個訂單（未來功能）

## 疑難排解

### 問題：還是出現頁首頁尾

**解決方案：**
1. 確認「頁首和頁尾」選項確實取消勾選
2. 清除瀏覽器快取後重試
3. 嘗試使用無痕模式測試

### 問題：QR Code 沒有顯示

**解決方案：**
1. 確認「背景圖形」已勾選
2. 檢查 Canvas 是否正確渲染（在螢幕上應該看得到）
3. 重新整理頁面後再試

### 問題：內容被裁切

**解決方案：**
1. 確認紙張大小符合模板設定
2. 檢查邊界是否設為「無」
3. 確認縮放比例為 100%
4. 在列印設定頁面調整元素位置

### 問題：出現空白頁

**解決方案：**
1. 確認邊界設為「無」
2. 檢查模板尺寸是否正確
3. 確認元素沒有超出模板範圍

## 推薦印表機設定

### 熱感應標籤機（推薦）

適合大量列印：
- **Brother QL 系列**（QL-700、QL-800）
- **Zebra GK/GX 系列**
- **DYMO LabelWriter**

優點：
- 無需墨水
- 快速列印
- 成本低

### 雷射印表機

適合辦公室環境：
- 支援自訂紙張大小
- 列印品質高
- 適合彩色列印

### 噴墨印表機

適合小量列印：
- 價格便宜
- 彩色效果好
- 注意墨水成本

## 常見問題

<details>
<summary>Q: 為什麼要關閉頁首頁尾？</summary>

A: 頁首頁尾會顯示網址、日期等資訊，佔用空間且不需要在訂單卡片上顯示。
</details>

<details>
<summary>Q: 可以同時列印多張訂單嗎？</summary>

A: 目前每次列印一張。如需批次列印，請重複列印動作，或等待未來的批次列印功能。
</details>

<details>
<summary>Q: 列印出來的大小不對怎麼辦？</summary>

A:
1. 確認印表機支援自訂紙張大小
2. 在印表機設定中建立自訂紙張尺寸（100mm × 50mm）
3. 確認縮放比例為 100%
</details>

<details>
<summary>Q: 可以改變卡片大小嗎？</summary>

A: 可以！前往[列印設定頁面](http://localhost:3000/print-settings)修改模板寬度和高度。
</details>
